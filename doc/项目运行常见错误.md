由上面的报错内容来看，主要时node-sass依赖python2这个程序，而我的电脑只有python3，所以找不到2，报错了。

针对这种问题：
1. 安装python2
2. 不装python2，安装node-sass的高级版本，这个项目好几年前的了。查了一下，可以用`sass`代替。



# Error message "error:0308010C:digital envelope routines::unsupported"
这个[参考](https://stackoverflow.com/questions/69692842/error-message-error0308010cdigital-envelope-routinesunsupported)，我是第四步使用`$env:NODE_OPTIONS = "--openssl-legacy-provider"`解决的。

# Proxy error: Could not proxy request /admin/auth/login from localhost:9528 to http://localhost:5678/.

这是没有配置后端地址，在`.env.development`中配置一下`VUE_APP_BASE_API`值就好。

> 这里配置之后，浏览没有反应，打开调试器发现触发`cors`问题，这个在后端服务里面配置上对应的中间件处理cors的函数即可。
